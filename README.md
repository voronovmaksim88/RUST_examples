# RUST_examples

Личные примеры и утилиты на языке Rust. Каждый каталог — отдельный пакет Cargo (в корне нет единого `workspace`).

## Проекты

### [`backup_1c`](backup_1c/)

Консольная утилита для резервного копирования папки базы **1С**: проверка путей, подготовка каталога с датой (`ГГГГ-ММ-ДД`), копирование через `fs_extra`. Пути к источнику и назначению хранятся в `config.json` рядом с исполняемым файлом (при первом запуске используются значения по умолчанию из кода). Цветной вывод (`colored`), на Windows включается поддержка ANSI в консоли (`winapi`).



---

### [`sctring_counter`](sctring_counter/)

Рекурсивный подсчёт **строк во всех файлах** относительно каталога, где лежит exe: для каждого файла печатается относительный путь и число строк, в конце — сумма. Пропускаются каталоги `.git`, `.venv`, `node_modules`, `__pycache__` и файлы `.xls` / `.xlsx`. На Windows завершение — по клавише **Esc**; на других ОС — Enter.



---

### [`Temperature_Converter`](Temperature_Converter/)

Простой **конвертер температур** в консоли: меню «°C → °F» или «°F → °C», ввод числа с клавиатуры, вывод с двумя знаками после запятой.



---

### [`test-makepad`](test-makepad/)

Минимальное **GUI на Makepad** (`makepad-widgets`): окно с числом и кнопкой «+1», счётчик увеличивается по клику. Демонстрация `live_design!`, `MatchEvent` / `handle_actions` и обновления текста у `Label`.


---

## Прочее

- **[`.skills`](.skills/)** — вспомогательные заметки (например, про цветной вывод в консоли Windows), не отдельный Rust-проект.

## Требования

- Установленный **Rust** и **Cargo** (`rustup`).
- Для сборки используются edition **2021** или **2024** в зависимости от пакета (см. `Cargo.toml` в каталоге проекта).
